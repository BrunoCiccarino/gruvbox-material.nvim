# gruverboxer-material.nvim

Gruverboxer Material for Neovim, This is a fork of the gruvbox.nvim theme. 

## Install:

- Lazy.Nvim:

```lua  
  {
    "BrunoCiccarino/gruverboxer-material.nvim",
    priority = 1000,
    config = true,
  },
```
